Beginner HTML & CSS Tags

HTML Tags

Html Tags can be categorized into container tags and empty tags.

Container Tags

Container tags have both an opening and closing tag. The text, numbers etc are all affected by the tag's purpose.
Tags such as <p> <h1> <pre> <code> <style> <em> <style> <div> are container tags.

Empty Tags

Empty tags also known as self-closing tags as they do not need a closing tag.
Tags such as <br> <hr> <img> <input> are empty tags.

Paragraph <p> Tag

<p> is used to create a single line of paragraph it can be used like this: 

<p>Hello this will be displayed in a single line </p>


Output:
    
Hello this will be displayed in a single line

List <li> & types of Lists

The items will be listed by numbers when using ordered list.

<ol>

</ol>

    
Output:

The items will be listed by bullet points when using unordered list.

<ul>

<li>Cherry</li>     
</ul>


Output:

  • Apple

CSS Tags

Background Color

background-color properties changes the background color of an element
<p style="background-color: pink;">The background of this element will be pink.</p>
Output:

The background of this element will be pink.

Border Style

The border-style properties is used to change the style of a border-style of an element
<p style="border-style: dotted;">The element wil have a dotted border.</p>
<p style="border-style: dashed;">The element wil have a dashed border.</p>
<p style="border-style: solid;">The element wil have a solid border.</p>
<p style="border-style: double solid solid double;">The element will have a mixed border.</p>
Output:

The element will have a dotted border.

The element will have a dashed border.

The element will have a solid border.

The element will have a mixed border.

Text Align

The text-align properties is used to change the horizontal alignment of text
<p style="text-align: center;">The element will be displayed in the center.</p>
Output:

The element will be displayed in the center.

Line Height

The line-height properties is used to changed the height of a line
<pre style="line-height: 30px;">↓This is 30px
↑Hello</pre>
<pre style="line-height: 15px;">↓This is 15px
↑Hello</pre>
Output: 
↓This is 30px
↑Hello

↓This is 15px
↑Hello

Text Decoration

The text-decoration property is used to change and or add decoration to a text
<pre style="text-decoration: underline red;">This will have a red underline.</pre>
<pre style="text-decoration: overline purple;">This will have a purple overline.</pre>
<pre style="text-decoration: line-through;">This will have a line cut through the text.</pre>
Output: 
This will have a red underline.
This will have a purple overline.
This will have a line cut through the text.

Color

The color property is used to change the color of the text
<pre style="color: black;">This text will be black.</pre>
<pre style="color: blue;">This text will be blue.</pre>
Output: 
This text will be black.
This text will be blue.

Letter Spacing

The letter-spacing property is used to change the amount of space between letters
<pre style="letter-spacing: 1px;">This text has a line spaceing of 1px.</pre>
<pre style="letter-spacing: 5px;">This text has a line spaceing of 5px.</pre>
<pre style="letter-spacing: 10px;">This text has a line spaceing of 10px.</pre>
Output: 
This text has a line spaceing of 1px.
This text has a line spaceing of 5px.
This text has a line spaceing of 10px.

Font Family

The font-family property is used to change font of a text or element
<pre style="font-family: Times New Roman;">This text font is Times New Roman.</pre>
<pre style="font-family: cursive;">This text font is cursive.</pre>
<pre style="font-family: fantasy;">This text font is fantasy.</pre>
Output: 
This text font is Times New Roman.
This text font is cursive.
This text font is fantasy.

Padding

The padding property is used to generate space inside an element's content
<pre style="padding-left: 50px; padding-top: 30px;">This text has been moved to the right by 50 px and moved to the bottom by 30px.</pre>

<pre style="text-align: right; padding-right: 66px; padding-bottom: 55px;">This text has been moved to the left by 66 px and moved to the top by 55px. While being align to the right</pre>
Output: 
This text has been moved to the right by 50 px and moved to the bottom by 30px.
This text has been moved to the left by 66 px and moved to the top by 55px.
While being align to the right

Margins

The margin propertie is used to create space around the outside of elements
<pre style="margin: 30px;">This text has a margin of 30px around all directions.</pre>
Output: 
This text has a margin of 30px around all directions.
